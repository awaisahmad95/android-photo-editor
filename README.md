# PhotoEditor

A Photo Editor library with simple, easy support for image editing using Paints, Text, Filters, Emoji, and Sticker-like stories.

## Features

- [**Drawing**](#drawing) on the image with the option to change its Brush's Color, Size, Opacity, and Erasing.
- Apply [**Filter Effect**](#filter-effect) on image using MediaEffect
- Adding/Editing [**Text**](#text) with option to change its Color with Custom Fonts.
- Adding [**Emoji**](#emoji) with Custom Emoji Fonts.
- Adding [**Images/Stickers**](#adding-imagesstickers)
- Pinch to Scale and Rotate views.
- [**Undo and Redo**](#undo-and-redo) for Brush and Views.
- [**Deleting**](#deleting) Views
- [**Saving**](#saving) Photo after editing.



## Benefits
- Hassle-free coding
- Increase efficiency
- Easy image editing

## Deleting
To delete a text, emoji, or image, we can click on the view to toggle the view highlighter box, which will display a close icon. So, by clicking on the icon, we can delete the view.
